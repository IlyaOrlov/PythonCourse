Переходы по страничкам сайта (кроме детальных страничек каждого фильма) выполнены ссылками в ф-ле base.html.

13.1 - добавлены странички:
- main_page.html - главная страничка (127.0.0.1:8000);
- premieres.html - страничка с премьерами (127.0.0.1:8000/premieres)
Данные, указанные на данных страничках, исключительно для примера.

13.2 - язык шаблонов django использован в:
- main_page.html (в ф-л передается список жанров, каждый элемент которого приводится
к прописной форме с помощью pipe - пример надуман);
- films.html;
- detail.html.

13.3:
- films.html - список всех фильмов с указанием только названий (127.0.0.1:8000/films);
- detail.html - подробная информация по конкретному фильму, если фильм с таким id не найден 
в базе - выводится ошибка (127.0.0.1:8000/films/film_id).

14.1 - добавлена таблица комментариев, связанная с таблицей фильмов.
Просмотреть все комментарии конкретного фильма можно на страничке фильма (127.0.0.1:8000/films/film_id),
там же можно добавить новый комментарий к данному фильму (изменения будут сохранены в базе и отображены
на этой же страничке).

14.2 - ограничение достапа к страничке new.html(127.0.0.1:8000/new), 
вход на нее только для администратора с :
login: superuser
pwd: cisco123
При попытке входа на эту страничку любого другого пользователя, он будет перенаправлен на страничку error.html - 
с разъяснением причины отказа.
